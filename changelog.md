# Changelog

В этом файле будут записываться все реализованные изменения в проекте.

## [Unreleased]
- **Оптимизация производительности (Zustand селекторы):** Полный рефакторинг доступа к состоянию плеера (`seek`, `duration`). Все компоненты переведены на использование атомарных селекторов, что радикально снизило нагрузку на CPU (устранены лишние перерендеринги при воспроизведении).
- **Регулятор громкости:** Реализован компонент `VolumeSlider` с поддержкой Mute, плавным управлением и интеграцией в глобальный стор.
- **Исправление багов (TrackList):** Исправлена ошибка дублирования ключей в списке треков, приводившая к нестабильности React дерева.
- **Архитектура плеера (Zustand):** Полный рефакторинг с использованием `zustand`. Удален `PlayerContext`; добавлено глобальное состояние.
- **Структура проекта (`src/`):** Миграция исходного кода в `src/`. Компоненты плеера распределены по изолированным модулям со своими локальными `*.module.css`.
- **Media Session API:** Логика вынесена в независимый хук `useMediaSession`, решающий проблемы пропадания сессии на мобильных устройствах.
- **Тексты песен:** Устранена зависимость `fetch` от длительности трека, починен скролл при переключении песен.
- **Цвета обложек:** Отделено извлечение цветов в кешируемый хук `useTrackColor` во избежание мерцания интерфейса.
- Созданы `roadmap.md` и `changelog.md` для отслеживания процесса и планирования рефакторинга.
